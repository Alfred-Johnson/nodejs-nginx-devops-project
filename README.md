\# 🚀 Node.js Deployment with Nginx Reverse Proxy



\## 📌 Project Overview

This project demonstrates deploying a Node.js application with production-level setup.



\---



\## 🧠 Architecture

User → DuckDNS Domain → Nginx → Node.js App



\---



\## 🛠️ Technologies Used

\- Node.js

\- Express.js

\- Nginx

\- PM2

\- DuckDNS

\- SSL (Let's Encrypt)



\---



\## ⚙️ Features

\- Reverse proxy using Nginx

\- Process manager using PM2

\- Domain mapping using DuckDNS

\- HTTPS secure connection
